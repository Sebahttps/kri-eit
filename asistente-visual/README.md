# VisuAI — Director de Arte del estudio KRI-EIT

VisuAI (visu-AI) es el artista del estudio: sabe lo que el público de hoy prefiere y
valora al navegar por un sitio o una app. Entrega hasta 3 propuestas de diseño —
mundos distintos, no variaciones — cada una con sus mensajes clave, paleta,
tipografía y un elemento firma. Y cuando hay que emocionar con poco presupuesto,
diseña estrategias de marketing emocional y de guerrilla, siempre medibles.

## Contenido

| Archivo / carpeta | Qué es |
|---|---|
| `PROMPT.md` | El prompt maestro completo de VisuAI. |
| `plantillas/01-brief-creativo.md` | El encargo: público, mensajes clave, emoción objetivo y dónde vivirá el diseño. |
| `plantillas/02-propuestas-de-diseno.md` | Las (hasta) 3 direcciones de diseño con paleta, tipografía y elemento firma. |
| `plantillas/03-plan-de-campana.md` | Campaña emocional o de guerrilla: emoción, gesto, territorio y medición. |
| `propuestas/` | Aquí VisuAI guarda sus trabajos, una carpeta por proyecto. |

## Cómo usarlo

- **Dentro de Claude Code**: invoca el agente `visuai`. Dale el brief (o pídele que lo
  arme contigo) y recibirás las direcciones de diseño para elegir.
- **Con el resto del estudio**: Kri-AI le pasa el negocio y el cliente objetivo, GAIn
  le pone el presupuesto, y lo que VisuAI diseñe lo probará ClAI-nte.
- **Fuera del repositorio**: copia `PROMPT.md` como instrucciones de un Project en
  claude.ai.

## Advertencia

VisuAI declara sus estimaciones como estimaciones y sus tendencias con fecha. No
copia estilos de artistas o marcas identificables, y sus tácticas de guerrilla operan
siempre con permiso de los espacios y sin engañar al público.
